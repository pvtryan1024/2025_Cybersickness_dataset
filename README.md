A dataset created from a 4-minute VR experience with a 0.5-second interval of Fast Motion Sickness scale inputs

Each csv file is composed of the following

| First Column  | Second Column | Third Column  | fourth Column | Fifth Column  | Sixth Column | Seventh Column  | Eighth Column | Ninth Column  | Tenth Column | Eleventh Column  |
| ------------- | ------------- | ------------- | ------------- | ------------- | ------------- | ------------- | ------------- | ------------- | ------------- | ------------- |
| Timestamp  |  FMS Score  | X axis Acceleration  | Y axis Acceleration  | Z axis Acceleration  | X axis Angular Velocity |  Y axis Angular Velocity  | Z axis Angular Velocity  |  Gender  | MSSQ score  | Age  | 


It is recommended to cut the data at the 420th row. The visual movement stops around that point.

The test environment has been requested multiple times; the video link is provided below.

https://drive.google.com/file/d/1KPVixU-WmrTpQA7QWYkSwJ8QLw2otMIC/view?usp=sharing

Caution! The resolution of the video is not what the user sees within the VR.

The dataset is from the work "Predicting Cybersickness Trend and Extent Based on FMS Labeled Dataset"
